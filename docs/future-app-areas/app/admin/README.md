# Admin Dashboard

This folder is reserved for the future Shopify admin dashboard.

Planned admin areas:

- Import delivery data from CSV or PDF-converted rows.
- Manage pincode groups.
- Manage product tag restrictions.
- Configure cutoff settings.
- Map Shopify shipping methods.
- Configure shipping hide rules.
- Configure shipping rename rules.
- Map Shopify payment methods.
- Configure payment hide rules.
- Configure validation messages.
- Run checkout rule simulations.

No admin code exists in this phase.
