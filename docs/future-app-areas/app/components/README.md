# Components

This folder is reserved for future reusable admin UI components.

Expected component areas:

- File import controls.
- Rule condition editors.
- Pincode group selectors.
- Product tag selectors.
- Shipping method mapping selectors.
- Payment method mapping selectors.
- Cutoff setting controls.
- Rule preview and simulator panels.

Components should display configured data clearly and avoid embedding operational values in source code.
