# Shipping Rename Rules Service

Future responsibility: manage configuration for renaming shipping methods.

## Responsibilities

- Reference source shipping method mappings.
- Store configurable rename labels.
- Support conditions based on pincode, area group, product tags, subscription classification, delivery group, and cutoff settings.
- Define priority when multiple rename rules could match.

## Constraint

Rename labels and source method names must come from imported data or admin configuration.
