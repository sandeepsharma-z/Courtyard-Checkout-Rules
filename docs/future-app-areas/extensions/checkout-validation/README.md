# Checkout Validation Extension

This folder is reserved for the future cart/checkout validation Function.

## Intended Scope

- Block checkout when configured pincode and product tag restrictions apply.
- Return configurable validation messages.
- Read published Shopify-side configuration.

## Out Of Scope For This Phase

- No generated Shopify Function code.
- No package files.
- No dependency installation.

## Configuration Rule

Pincode groups, product tags, and customer-facing error messages must come from admin configuration or imported data.
