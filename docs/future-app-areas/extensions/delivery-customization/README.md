# Delivery Customization Extension

This folder is reserved for the future Shopify delivery customization Function.

## Intended Scope

- Hide shipping methods based on configured shipping hide rules.
- Rename shipping methods based on configured shipping rename rules.
- Read published Shopify-side configuration.
- Evaluate checkout pincode, product tags, area group, cutoff settings, and mapped shipping methods.

## Out Of Scope For This Phase

- No generated Shopify Function code.
- No package files.
- No dependency installation.

## Configuration Rule

Shipping method names, rename labels, cutoff times, pincode groups, product tags, and delivery labels must come from imported data or admin configuration.
