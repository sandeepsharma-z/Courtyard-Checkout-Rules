# Payment Customization Extension

This folder is reserved for the future Shopify payment customization Function.

## Intended Scope

- Hide payment methods based on configured payment rules.
- Read published Shopify-side configuration.
- Evaluate pincode groups, selected shipping method conditions, product tags, and cart/order conditions where supported.

## Out Of Scope For This Phase

- No generated Shopify Function code.
- No package files.
- No dependency installation.

## Configuration Rule

Payment method names and matching conditions must come from admin configuration or imported data.
