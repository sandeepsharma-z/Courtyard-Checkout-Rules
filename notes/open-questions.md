# Open Questions

These questions should be answered before implementation.

- What BSure rules are currently active?
- Are BSure rules exportable, or must they be captured from screenshots?
- Which Shopify shipping methods currently exist in the store?
- Which Shopify payment methods currently exist in the store?
- Which product tags currently drive restrictions?
- Which delivery data source is authoritative?
- Are pincode rules one row per pincode, or can multiple rows apply to the same pincode?
- What should happen when imported delivery data conflicts with manual admin configuration?
- Should rule changes support draft and published states?
- What simulator scenarios are required for approval before production launch?

Do not answer these questions with hardcoded values in code. Capture answers as imported data or admin-managed configuration.
